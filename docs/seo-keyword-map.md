# aama.io — Organic Traffic Strategy & Keyword → URL Map

> Working document for the SEO/content roadmap. Phasing: **Singapore → APAC → Global.**
> **Positioning category (deck/narrative, NOT a search term):** *End-to-End Fund Operating Platform / "Fund OS"*. Nobody searches this — keep it for pitch/brand copy only.
> **SEO surfaces lead with how buyers actually search:** *fund administration software Singapore*, *fund accounting software*, *LP portal for fund managers*. Primary anchor: **Fund Administration Software Singapore**. Titles, meta descriptions, H1s and section eyebrows must use query language, not category language.

## Strategic thesis (validated against live SERPs, June 2026)

The head term **"fund administration software Singapore"** is a crowded, mixed-intent SERP owned by service firms (IQ-EQ, Lanturn, BoardRoom, CSC) and entrenched software (Carta APAC, RAISE, Dynamo). It is a long, slow fight and half the searchers want to *outsource*, not buy software.

The winnable, high-value traffic is **one step upstream**: the Singapore fund-formation & regulatory cluster, and the calculator/tool cluster — where the ICP (emerging GPs, family offices, fund admins) searches *before* they need a fund-ops platform, and where the competition (Auptimate, microsites) is beatable. These are **low-volume, high-value B2B terms**: optimise for intent and demo conversion, not impressions.

Leverage points we already own: **13 free calculators**, a now server-rendered **blog**, `llms.txt` + AI-crawler allowances for **AEO**.

## Finalized keyword hierarchy (decisive picks)

**Anchors**
1. **`fund administration software Singapore`** — #1 primary anchor. Exact buyer intent, geo-defensible, both tracks search it. *(live: homepage title/H1/desc)*
2. **`private equity fund administration software`** — #2 **global** anchor. Excludes mutual-fund/retail noise, high commercial intent; the comparison roundups ranking for it (Allvue, Carta, Juniper Square) are beatable with genuinely deeper content. → **needs a dedicated page** (`/solutions/private-equity-fund-administration` or `/pe-fund-administration-software`).
3. **`venture capital fund administration software`** — sister page. Lower volume, but the **VCC angle makes us near-uncontested in Singapore**. → dedicated VC page.

**Two-track supporting cluster** (mind the subject flip)
- **Track 1 — fund admin firms (operators):** `software for fund administrators`, `fund administrator software`. Signals an *admin firm buying its own ops platform*, not a GP shopping for an administrator. Small volume, almost nobody writes for it — the white-label / Capital-Engine story. → `/solutions/fund-administrators` (white-label).
- **Track 2 — GP founders:** `fund administration for emerging managers`, `in-house fund administration software`. The "should I self-administer my first fund?" searcher — highest-leverage long-tail buyer. → pillar content + the emerging-manager page (content already partly exists in `/blog/how-to-start-a-fund-singapore`).

**Feature-intent (tool-led, mid-evaluation, high-convert)** — each justifies a calculator/tool as the ranking asset:
- `capital call software` → `/tools/capital-call-schedule`
- `distribution waterfall software` → `/tools/waterfall` (the Waterfall Engine)
- `NAV calculation software` → new tool/feature page
- `LP portal software` → LP-portal module page

**Singapore moat long-tail** (tiny individually, compounding topical authority — nobody else can write these with real regulatory depth):
- `VCC fund administration`, `VCC sub-fund accounting` → new pages (pair with the shipped VCC pillar)
- ⚠️ `MAS RFMC compliance software` — **RFMC closed to new applicants 1 Aug 2024.** Don't target it as a live regime. Instead target `MAS LFMC compliance software` / `VCFM compliance`, **and** consider an `RFMC → LFMC transition` page to capture residual "RFMC" searches and redirect them (high regulatory-depth signal).

**Excluded — do NOT target:** `custom fund administration software`. Attracts bespoke-development / agency shoppers and invites scope-creep conversations, not SaaS buyers. aama.io is a **configurable product, not a custom build** — keep "custom" out of the keyword set and the copy entirely.

## Geographic phasing

- **Phase 1 (0–6 mo) — Singapore:** Clusters A + B + optimise the 13 tool pages. Own VCC / 13O-13U / licensing / SPV education + calculators.
- **Phase 2 (6–18 mo) — APAC:** Replicate Cluster A for adjacent jurisdictions — **Hong Kong OFC** (VCC analogue), **India AIF** (Cat I/II/III), **Malaysia/Labuan**, **Indonesia**. Add APAC modifiers to category terms.
- **Phase 3 (18 mo+) — Global:** Compete on evergreen Cluster B + category terms without geo modifier; emerging-manager content for US/UK. Lead with IFRS 9 / mid-market angle, not head-to-head on "fund admin software."
